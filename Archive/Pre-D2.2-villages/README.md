# Pre-D2.2 Village Sources (Archived 2026-06-07)

This folder contains pre-D2.2 sources for village mechanical perks.

## Why archived

`Vilas e Bônus.md` was the original master list. It drifted from the
post-Skill-Reform `Mechanics/Character_Options/Villages/*.md` individual
files. D2.2 hand-curated the canonical version into structured YAML
under `Mechanics/Character_Options/Villages/*.yaml`; the master file
is no longer authoritative.

See `docs/superpowers/specs/2026-06-07-d2.2-village-traits-design.md`
§ 2 for the canonical table.
