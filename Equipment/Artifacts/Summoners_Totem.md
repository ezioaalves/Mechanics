# Summoner's Totem [Sealed Item]

This totem is a small object that represents the spirit of a specific blood pact.

**Sealed Item (totem)**
*   Minor (Lv 6): DC 17

**Property:** This item functions like a greater summoning scroll, but using this object does not increase the time to perform a technique or become inert after use.
**Special:** This sealed item must be crafted for a specific blood pact, and is not interchangeable.
