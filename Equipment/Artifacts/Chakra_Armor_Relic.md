# Chakra Armor [Minor Relic]

This medium armor protect the user against most techniques and lethal blow.

**Minor Relic (medium armor)**
*   Armor Bonus: +4
*   Max Dex: +4
*   ACP: 0
**Power:** As long as Chakra Pool is not depleted, wielder can **Fly** at base speed (poor).
**Property:** +4 deflection bonus against ranged attacks. Maximum Chakra Pool +12.
**Property:** Protects against any *Chakra Control, Genjutsu, or Ninjutsu* technique if it can store the entire Chakra Cost. Stores up to 150 points before breaking. Techniques of Rank 7+ bypass this.
