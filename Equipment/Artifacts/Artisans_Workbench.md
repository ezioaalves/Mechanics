# Artisan's Workbench [Sealed Item]

This workbench is fully stock and covered with inscriptions of strange symbols.

**Sealed Item**
*   Greater (Lv 12): +2 DC 35

**Enhancement:** Craft checks (competence; see text)
**Power (5 charges, see text):** You can expend a charge from the artisan's workbench to create objects as though you had the Mastercrafter feat. If you have the Specialty class feature, you can expend a charge instead to create items as though you had the Double Order specialty.
**Special:** The bonus to Craft checks applies to only one craft skill, determined when the item is created. When all the charges are expended, the workbench still provides a bonus to Craft checks when used.
