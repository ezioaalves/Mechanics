# Lifestyle

*Source: NarutoD20 Redux v1.3, Chapter 7 – Equipment (p74). Additional Housing items. See D20 Modern p122 for more information on Housing.*

| Lifestyle Item | Cost |
| :--- | :--- |
| Manor | 40 |
| Castle | 44 |

- A **Manor** can have up to ten bedrooms and five suites, with the appropriate number of living and working spaces and a reasonable patch of surrounding land.
- A **Castle** can have thirty bedrooms, fifteen suites, an appropriate number of living and work spaces and an expansive patch of surrounding land.
