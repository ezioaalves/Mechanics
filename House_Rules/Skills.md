# 📜 FINAL SKILL LIST 

All characters use the **skill point buy system from **<span class="whitespace-normal"><strong data-start="229" data-end="298">Pathfinder</strong></span>

 for skill progression. At each level, characters receive a number of skill points based on their class and Intelligence modifier, which are then invested directly into skills. All skills are treated as class or cross-class according to their respective class definitions, following the standard Pathfinder rules for maximum ranks and scaling.

Additionally, the skill **Knowledge (Arcana)** is expanded to incorporate all functions previously associated with **Knowledge (Ninja)**. This includes understanding chakra theory, ninja traditions, jutsu classification, and related subjects. As a result, **Knowledge (Ninja) is removed as a separate skill**, and any references to it should be treated as **Knowledge (Arcana)** instead.

**Ninjutsu (Int)**<br data-start="1408" data-end="1411">

**Taijutsu (Str or Dex)**<br data-start="1584" data-end="1587">
