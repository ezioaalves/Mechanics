# Sealweaver

**Prerequisite**: Int 13.

**Benefit**: You can learn Fuinjutsu techniques normally.

If selected at 1st level, you gain Fuinjutsu as a class skill whenever Craft (calligraphy) is a class skill.

**Normal**: A character without this feat takes a -8 penalty to Learn checks and takes twice the normal amount of time to learn Fuinjutsu techniques.
