# Innate Control [Meta-Chakra]

You are able to manifest true mastery over chakra.

**Prerequisite**: Chakra Control 15 ranks, Harmony

**Benefit**: The character is able to use the Kinobori, Tadayou or Yukigutsu technique once per round as a reaction, without needing to make a perform check, to concentrate or use chakra, even if the character does not know the techniques. Every use expends a meta-chakra charge.
