# Class Plus

Your skills in a particular class are unmatched.

**Benefit**: You gain two talents from an advanced or prestige class in which you have levels already in. For example, classes that offer a reoccurring same named benefit with different options qualify, such as the Exarch's Exarch Arcana or Genjutsu Master's Genjutsu Mastery. You must still qualify for the abilities you select as normal in all forms. The restrictions for those classes still apply (i.e. if the class says you can only take it once this feat does not remove that limitation. If a feature can be taken twice you may use both uses of this feat to gain it twice).

**Special**: This does not apply to bloodline classes. Talents that grant bonus feat instead of a special ability cannot be selected.

You may select this feat multiple times.
