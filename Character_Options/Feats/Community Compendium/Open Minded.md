# Open Minded

Born with a truly gifted brain, the character with this feat has the capability to be skilled in many things.

**Prerequisite**: Int 15.

**Benefit**: The character gets an extra skill point per HD; this feat can be taken multiple times. If he gains another HD, he gains another skill point for each time he possesses this feat.

**Special**: If taken at level 1, this extra skill point is multiplied by 4 as usual.
