**Sarutobi Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, she receives a +1 competence bonus on checks using that skill: Bluff, Chakra Control, Craft (calligraphy), Concentra tion, Decipher Script, Diplomacy, Gamble, Gather Information, Genjutsu, Intimidate, Investigate, Listen, Sleight of Hand, Spot 
Feat: Select one of the following: Advanced Bloodline (Seishintoku), Educated, Force of Personality, Genjutsu Adept, Genjutsu Technique Focus (Phantasm, Doujutsu or Mind -Affecting), Skilled, Will over Flesh 

**Affiliation:** You require 1 less success to learn Seishin Hijutsu techniques. 

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
