The Seijina are a legendary clan, said to be descended from the lightning god Raijin himself. They possess the "Raijinsatsu" bloodline and the "Purumazuton" (Plasma Release) advanced nature transformation, allowing them to create and control super-heated plasma, a state of matter even more destructive than lightning. Their power is immense and terrifying to behold.

**Seijina Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Concentration, Intimidate, Knowledge (physical sciences), Ninjutsu, Taijutsu.

**Feat:** Select one of the following: Advanced Bloodline (Raijinsatsu), Elemental Specialization (Plasma), Ninjutsu Adept, Genin, Empower Technique.

**Affiliation:** You require 1 less success to learn Seijina Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +2
