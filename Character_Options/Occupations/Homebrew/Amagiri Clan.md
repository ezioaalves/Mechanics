The Amagiri are a formidable clan possessing the Boil Release (Futton) advanced nature transformation, allowing them to create a highly corrosive vapor capable of melting nearly anything. Their unique "Ajna Bindu," a third-eye marking that appears on their forehead when their power is active, is a symbol of the immense chakra control required to master this dangerous ability.

**Amagiri Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Concentration, Intimidate, Ninjutsu, Survival.

**Feat:** Select one of the following: Advanced Bloodline (Ajna Bindu), Elemental Specialization (Boil), Ninjutsu Adept, Genin, [Universal: Risky Strike].

**Affiliation:** You require 1 less success to learn Amagiri Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
