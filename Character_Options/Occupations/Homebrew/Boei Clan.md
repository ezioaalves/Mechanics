The Boei clan are the ultimate protectors, possessing the "Oukokoi" (Royal Guard) bloodline which grants them unparalleled defensive capabilities. They can create powerful chakra barriers, reinforce their bodies to withstand incredible damage, and sense incoming threats with uncanny accuracy. They are the living shields of the ninja world.

**Boei Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Concentration, Ninjutsu, Sense Motive, Taijutsu.

**Feat:** Select one of the following: Advanced Bloodline (Oukokoi), [Universal: Defensive Stance], Endurance, Genin, Great Fortitude.

**Affiliation:** You require 1 less success to learn Boei Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
