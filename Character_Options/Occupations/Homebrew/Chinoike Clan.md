The Chinoike clan was a powerful clan once feared for their Ketsuryugan, a dojutsu that grants them the ability to manipulate any liquid with a high iron concentration, most notably blood. This makes them terrifying masters of genjutsu and ninjutsu, capable of controlling opponents from within or creating powerful, blood-based attacks.

**Chinoike Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Genjutsu, Intimidate, Knowledge (ninja lore), Ninjutsu.

**Feat:** Select one of the following: Advanced Bloodline (Ketsuryugan), Genjutsu Adept, Ninjutsu Adept, Genin, Iron Will.

**Affiliation:** You require 1 less success to learn Chinoike Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
