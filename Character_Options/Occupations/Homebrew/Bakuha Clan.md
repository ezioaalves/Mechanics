The Bakuha clan possesses the "Genshigan," a terrifying dojutsu that allows them to perceive the molecular structure of matter and, with extreme focus, break it down. This makes them incredibly dangerous shinobi, capable of disintegrating objects and living beings. This power requires immense concentration and chakra control to use effectively and safely.

**Bakuha Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Concentration, Knowledge (physical sciences), Ninjutsu, Taijutsu.

**Feat:** Select one of the following: Advanced Bloodline (Genshigan), Chakra Penetration, Ninjutsu Adept, Genin, Focused.

**Affiliation:** You require 1 less success to learn Bakuha Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
