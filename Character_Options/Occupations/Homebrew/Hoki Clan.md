The Hoki clan is a branch family of the prestigious Hyuuga clan. They possess the "Toushigan" (Telescope Eye), a dojutsu that, while not as powerful in all aspects as the Byakugan, grants them incredible long-range vision and the ability to see through solid objects. This makes them exceptional scouts, spies, and archers.

**Hoki Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Investigate, Ninjutsu, Search, Spot, Survival.

**Feat:** Select one of the following: Advanced Bloodline (Toushigan), Far Shot, Keen Senses, Genin, Precise Shot.

**Affiliation:** You require 1 less success to learn Hoki Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
