The Sendo clan is a noble family of diplomats and strategists. They possess the "Keireigan," a subtle dojutsu that allows them to perceive the emotional state and true intentions of others. This makes them masters of negotiation, interrogation, and psychological warfare, able to navigate complex social and political landscapes with ease.

**Sendo Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Bluff, Diplomacy, Gather Information, Genjutsu, Knowledge (behavioral science, civics), Sense Motive.

**Feat:** Select one of the following: Advanced Bloodline (Keireigan), Force of Personality, Combat Tactics, Genin, Trustworthy.

**Affiliation:** You require 1 less success to learn Sendo Hijutsu techniques.

**Wealth Bonus Increase:** +2

**Reputation Bonus Increase:** +1
