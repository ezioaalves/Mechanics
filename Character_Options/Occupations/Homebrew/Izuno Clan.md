The Izuno clan possesses the "Seiseigan" (Star Eye), a beautiful and powerful dojutsu that allows them to perceive chakra pathways as a celestial star chart. This gives them incredible insight into the flow of chakra, allowing them to predict an opponent's moves, identify weaknesses, and even track the residual chakra of past events with stunning accuracy.

**Izuno Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Genjutsu, Investigate, Ninjutsu, Sense Motive.

**Feat:** Select one of the following: Advanced Bloodline (Seiseigan), [Universal: Defensive Stance], Insightful Reflexes, Genin, Studious.

**Affiliation:** You require 1 less success to learn Izuno Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
