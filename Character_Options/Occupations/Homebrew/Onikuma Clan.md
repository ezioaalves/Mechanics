The Onikuma are a legendary and mysterious clan, whispered to possess the "Tokeidome" (Time Stop) bloodline. This ultimate power allows them to halt the flow of time for a mere instant, a moment that can decide any battle. This ability is incredibly rare, difficult to master, and drains an immense amount of chakra, making its users both revered and feared.

**Onikuma Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Concentration, Genjutsu, Ninjutsu, Sense Motive.

**Feat:** Select one of the following: Advanced Bloodline (Tokeidome), Focused, Iron Will, Genin, Improved Chakra Pool.

**Affiliation:** You require 1 less success to learn Onikuma Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +2
