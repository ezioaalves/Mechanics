The Rinha clan possesses the "Katsugan" (Living Eye), a dojutsu that allows them to perceive the flow of life force and chakra within living organisms. This unique ability makes them exceptional medics, trackers, and analysts, able to diagnose injuries and track targets with uncanny precision.

**Rinha Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Knowledge (earth and life sciences), Ninjutsu, Search, Survival, Treat Injury.

**Feat:** Select one of the following: Advanced Bloodline (Katsugan), Medical Expert, Keen Senses, Genin, Studious.

**Affiliation:** You require 1 less success to learn Rinha Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
