The Kumanoi clan possesses the "Shikawa Myaku" (Antler Vein), a bloodline that allows them to grow and manipulate their bone structure, similar to the Kaguya clan's Shikotsumyaku. However, the Kumanoi's bone manifestations often resemble the powerful antlers of deer and moose, which they use for devastating charging attacks and incredibly durable defenses.

**Kumanoi Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Intimidate, Ninjutsu, Survival, Taijutsu, Tumble.

**Feat:** Select one of the following: Advanced Bloodline (Shikawa Myaku), [Universal: Risky Strike], Endurance, Genin, Powerful Maneuvers.

**Affiliation:** You require 1 less success to learn Kumanoi Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
