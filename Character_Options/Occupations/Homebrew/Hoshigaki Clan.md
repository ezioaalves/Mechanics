The Hoshigaki clan is a fearsome clan from Kirigakure, whose members possess distinct shark-like features, including sharp teeth and gills. They are powerful shinobi with a natural affinity for Water Release and a massive chakra pool. Their aggressive, predatory combat style makes them formidable opponents, especially in aquatic environments.

**Hoshigaki Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Intimidate, Ninjutsu, Survival, Swim, Taijutsu.

**Feat:** Select one of the following: Genin, Ninjutsu Adept, [Universal: Risky Strike], Improved Chakra Pool, Archaic Weapon Proficiency.

**Affiliation:** You require 1 less success to learn Hoshigaki Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
