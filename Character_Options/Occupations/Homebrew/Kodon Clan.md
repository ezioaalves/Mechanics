The Kodon clan possesses the "Ketsueki" (Blood) bloodline, a unique and macabre ability that allows them to manipulate their own blood after it leaves their body. They can harden it to create sharp weapons, lay traps, or use it in its liquid form for binding and tracking purposes. This makes them resilient and unpredictable fighters.

**Kodon Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Ninjutsu, Survival, Taijutsu, Treat Injury.

**Feat:** Select one of the following: Advanced Bloodline (Ketsueki), Great Fortitude, Ninjutsu Adept, Genin, Endurance.

**Affiliation:** You require 1 less success to learn Kodon Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
