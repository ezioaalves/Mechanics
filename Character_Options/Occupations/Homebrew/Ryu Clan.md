The Ryu clan are known for their "Kyome" (Giant Roar) bloodline, which allows them to generate powerful, concussive sound waves from their mouths. Unlike the subtle genjutsu of the Taketori, the Ryu clan's techniques are pure, raw power, capable of shattering stone, disorienting opponents, and creating devastating shockwaves.

**Ryu Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Concentration, Intimidate, Ninjutsu, Survival.

**Feat:** Select one of the following: Advanced Bloodline (Kyome), Ninjutsu Adept, [Universal: Risky Strike], Genin, Great Fortitude.

**Affiliation:** You require 1 less success to learn Ryu Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
