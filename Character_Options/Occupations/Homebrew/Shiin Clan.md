The Shiin clan are masters of shadow manipulation, possessing the "Koimeigan," a bloodline that allows them to control and solidify darkness itself. Unlike the Nara clan's shadow imitation techniques, the Shiin can create tangible weapons, armor, and constructs from shadows, making them formidable and unpredictable combatants who thrive in the dark.

**Shiin Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Craft (mechanical), Hide, Move Silently, Ninjutsu, Taijutsu.

**Feat:** Select one of the following: Advanced Bloodline (Koimeigan), Ninjutsu Adept, Stealthy, Genin, [Universal: Risky Strike].

**Affiliation:** You require 1 less success to learn Shiin Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
