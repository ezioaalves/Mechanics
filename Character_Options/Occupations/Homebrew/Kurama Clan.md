The Kurama clan are unparalleled masters of genjutsu. Their bloodline grants them such incredible talent in this art that their illusions can have real-world consequences, convincing the victim's brain to manifest physical damage. This power, however, can sometimes be unstable, creating a dangerous second personality within the user if not properly controlled.

**Kurama Clan (Occupation)**

**Prerequisite:** Age 8+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Bluff, Chakra Control, Craft (painting), Genjutsu, Knowledge (behavioral science).

**Feat:** Select one of the following: Advanced Bloodline (Kurama), Genjutsu Adept, Technique Focus (Genjutsu), Genin, Force of Personality.

**Affiliation:** You require 1 less success to learn Kurama Hijutsu techniques.

**Wealth Bonus Increase:** +1

**Reputation Bonus Increase:** +1
