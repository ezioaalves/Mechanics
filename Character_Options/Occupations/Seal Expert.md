You come from a background of expertise in sealing ninjutsu. 

**Prerequisite:** Age 15+.

**Skills:** Select any two skills from the following list as permanent class skills. If a skill the character selects is already a class skill, he receives a +1 competence bonus on checks using that skill: Chakra Control, Craft (calligraphy), Fuinjutsu, Knowledge (ninja lore), Ninjutsu. 

**Feat:** The seal expert gains the Sealweaver feat.

**Reputation Bonus Increase:** +1.
