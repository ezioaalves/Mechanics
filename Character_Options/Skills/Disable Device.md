# Disable Device

**Ability:** Dexterity (Dex)
**Trained Only:** Yes
**Armor Check Penalty:** Yes

Disable Device is the skill of picking locks, disabling traps, sabotaging mechanisms, and bypassing security systems. *Naruto D20 has no entry for this skill in the "NEW USES FOR EXISTING SKILLS" section — the unmodified Pathfinder rules apply, with d20 Modern preserved as the more granular reference.*

---

## Description

A character with Disable Device can open locked doors, defuse explosives, sabotage equipment so it fails at a chosen moment, and disarm both mechanical and magical traps. In Kaihou's chakra-rich world, "device" can include sealed mechanisms whose triggers are chakra-based — the GM may rule that some traps require both Disable Device *and* Fuinjutsu.

---

## Check

### Lock Quality *(Pathfinder + d20 Modern merged)*

| Lock Quality (example) | DC |
|---|---|
| Simple (briefcase lock, cheap home deadbolt) | 20 |
| Average (home deadbolt) | 25 |
| Good / High-quality (business deadbolt) | 30 |
| High security (bank vault alarm) | 35 |
| Amazing / Ultra-high security (bank headquarters vault) | 40 |

Without proper tools (lockpicks for mechanical, electrical tool kit for electronic), **+10** to the lock DC. *(d20 Modern's "no tools = −4 penalty" is replaced by PF's "no tools = +10 DC" per priority — same direction, more consequential.)*

### Disable a Device

| Device | DC |
|---|---|
| Simple device | 10 |
| Tricky device | 15 |
| Difficult device | 20 |
| Extremely complex device | 25 |

### Magic Traps *(Pathfinder)*

- **DC = 25 + the spell level of the magic used to create the trap.**

### Security Devices *(d20 Modern preserved as practical reference)*

| Device (example) | DC |
|---|---|
| Cheap (home door alarm) | 20 |
| Average (store security camera) | 25 |
| High quality (museum motion detector) | 30 |
| High security (bank vault alarm) | 35 |
| Ultra-high security | 40 |

To disable a *monitored* device **without alerting the monitor**, the attempt requires an additional **10 minutes** and an electrical tool kit, and increases the DC by **+10**.

### Outcomes

- **Success:** Device disabled.
- **Failure by ≤4:** No progress; you may try again.
- **Failure by ≥5:** Something goes wrong — a trap springs; sabotage appears successful but the device still works normally; an alarm trips.

### Sabotage

You can rig a simple device to work normally for a while and then fail (typically after **1d4 rounds or minutes** of use).

---

## Action

- **Simple device or lock:** Full-round action.
- **Tricky device:** 1d4 rounds.
- **Difficult device:** 2d4 rounds.
- **Intricate or complex devices** (e.g. Foundry sealed mechanism, multi-tag chakra trap): 2d4 minutes or as the GM rules.

---

## Try Again

Yes, provided you know you have failed. If a failure by ≥5 sprang the trap or caused the sabotage to fail invisibly, you may not realise a retry is needed.

---

## Special

- You can **take 10** on a Disable Device check. You can **take 20** to open a lock or disable a security device — unless you're trying to prevent your tampering from being noticed.
- A character with the **Deft Hands** feat (Pathfinder) gains **+2** on Disable Device; **+4 at 10+ ranks**. *(Pathfinder threshold — no −3 conversion.)*
- A character with the **Cautious** feat (d20 Modern) and **at least 1 rank** in Disable Device gains **+2** on the skill.
- Rogue / Ninja Hunter / Ninja Police characters and others with **Trapfinding** can disarm magic traps (Pathfinder); without it, Disable Device on a magic trap fails automatically.
- Without the appropriate kit, take **−4** on the check (d20 Modern preserved as a softer alternative to PF's +10 DC for missing tools — GM may use whichever is harsher for the situation).

## Synergy

*(Naruto D20 / d20 Modern synergy thresholds converted via the −3 rank-prereq rule. Original value was "5 ranks". The d20 Modern source skills "Craft (electronic)" and "Craft (mechanical)" are both treated as subcategories of Pathfinder **Craft** in Kaihou's converted list.)*

- A character with **2 or more ranks** in the appropriate **[Craft](Craft.md)** subcategory (mechanical for mechanical devices; the GM may rule an electronic-flavoured Craft subcategory for electronic devices in modern-tech settings) gains a **+2 bonus** on Disable Device checks **against devices of that type**. *(Naruto D20.)*

## Source Notes

*Priority: Naruto D20 > Pathfinder > d20 Modern.*

- **Naruto D20 (Disable Device (Dex).md, vault):** Craft-subcategory synergies (Craft (electronic) → +2 against electronic devices; Craft (mechanical) → +2 against mechanical devices). Rank threshold "5" converted to **2** via the −3 rule. Naruto D20's PDF has no "NEW USES FOR EXISTING SKILLS" entry for Disable Device — the base d20 Modern rules apply unmodified except for the synergy notes.
- **Pathfinder ([d20pfsrd.com/skills/disable-device](https://www.d20pfsrd.com/skills/disable-device/)):** Backbone — Dex-based, trained-only, DC tables for locks and devices, magic-trap DC formula, Trapfinding interaction, Deft Hands scaling at 10+ ranks. **Naruto D20 uses Int** for this skill in the d20 Modern–derived stat block (since Naruto D20 inherits d20 Modern's Disable Device as Int-based); Pathfinder switched to Dex. **PF wins per priority** — Disable Device in Kaihou is **Dex**, not Int. This is a meaningful character-build shift; review existing PCs.
- **d20 Modern (modernsrd.pdf p. 70):** The detailed lock-quality and security-device tables (Cheap → Ultra-High Security), the "disable monitored device without alerting" rules (additional 10 min + electrical tool kit + DC 10), the rig-for-delayed-failure use, the Cautious feat. Preserved as practical references.

**Conflict resolution:** **Ability:** Pathfinder (Dex) wins over d20 Modern (Int). **Tools-missing penalty:** PF says +10 DC; d20M says −4 on roll. Both preserved (different magnitudes; GM picks per situation). **DC ranges:** PF and d20M use slightly different category labels but overlap directly; the merged table covers both vocabularies.
