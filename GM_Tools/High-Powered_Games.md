# High-Powered Games

*Source: NarutoD20 Redux v1.3, Character Creation (p9). An optional set of GM toggles for running a very high-powered campaign.*

If you wish to run a very high-powered campaign, consider the following suggestions:

- Use a **40 point-buy** system, or roll 5d6 per ability score, rerolling ones and discarding the two lowest.
- Have at least **1 Power Unit** (GM's discretion; see [[Power Units]]).
- **Halve** the time required to Learn and Master techniques.
- Use the variant rules for run speed and unarmed damage.
- Have characters start with **1d4 additional techniques**, or increase training days at creation by 50%.
- Grant **Balance, Chakra Control, Climb, Disguise, Genjutsu, Hide, Listen, Move Silently, Ninjutsu, Search, Spot, Survival and Taijutsu** as permanent class skills to all characters.
- Let characters **recover Chakra twice as fast.**

> Note (Kaihou): the standard recommended build is a 32- or 36-point buy. These toggles stack on top of normal character creation. Skill names above are the original Redux list — under the Kaihou skill conversion, several map to consolidated Pathfinder-shaped skills (e.g. Balance/Tumble → Acrobatics, Hide/Move Silently → Stealth, Listen/Spot → Perception). See [[Skill - Conversion]].
