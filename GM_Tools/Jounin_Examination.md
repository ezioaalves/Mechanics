# Jounin Examination

A 6-part individual exam to test if a shinobi is ready for the rank of Jounin. Candidates need **102 points** to pass.

## The Parts

### 1. The Scroll Test (72 Hours)
*   **Task:** Seal as many elemental techniques as possible using *Youso Fuuin no Jutsu*.
*   **Scoring:** 1 point per rank of the sealed technique.
*   **Requirement:** 15 points to pass.
*   **cheating:** Disqualified if signature doesn't match.

### 2. The Ninja Paper Test (90 Minutes)
*   **Task:** Answer 25 questions about the Ninja World.
*   **Checks:** Knowledge (ninja lore).
*   **Scoring:** 23 points max. 15 points to pass.
*   **Time:** Halve time by taking a -10 penalty to check.

### 3. The Efficiency Test (10 Days)
*   **Task:** Execute solo missions to accumulate points.
*   **Requirement:** 35 points to pass.
*   **Points:** D (2), C (5), B (15), A (20), S (35).

### 4. The First Combat Test (Non-lethal)
*   **Enemy:** 1 Rookie Jounin and 3 Genins.
*   **Restrictions:** No Templates or Advanced Bloodlines (except *Moujuu Aishou*).
*   **Requirement:** Begin with 20 points, lose 1 per round. Must have at least 1 point to pass.

### 5. The Second Combat Test (Non-lethal)
*   **Enemy:** 2 Mid-level Chuunins.
*   **Requirement:** Same point system as Part 4. All abilities allowed.

### 6. The Finale (Lethal)
*   **Enemy:** 1 Elite Jounin.
*   **Scoring:** Begin with 1 point, gain 1 per round. 10 points to pass.
*   **Bonus:** +20 points if the opponent is defeated.

## Point Equivalencies
| Points | Rank Equivalence |
| :---: | :--- |
| < 15 | Genin |
| 16-101 | Chuunin |
| 102+ | Jounin |
| 135+ | ANBU |
| 150+ | Hunter-Nin |
